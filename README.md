# Desafío IACC Backend

## Requerimientos

- [docker](https://www.docker.com/products/docker-desktop/)

## Instalación

```bash
$ npm install
```

## Build

```bash
# dev
$ npm run start

# watch mode
$ npm run start:dev

# prod
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
